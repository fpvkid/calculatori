Watch out for the "Nap"

I genuinely hate it when I find a perfect fabric, start cutting, and realize halfway through that the pattern only looks "right" in one direction. This is called the "nap." If you’re using velvet, corduroy, or any fabric with a directional print, you can’t just flip pieces upside down to save space. You’ll need to buy about 20% more yardage to make sure everything faces the same way, or your finished chair will look like a patchwork mess.

Upholstery vs. Apparel width

Most beginners don’t realize that fabric comes in different widths. If you’re buying quilting cotton (44"), you’ll need a lot more length than if you’re buying 54-inch upholstery fabric. Always check the width of the bolt before you order. This calculator defaults to 54 inches because that’s the gold standard for furniture.
